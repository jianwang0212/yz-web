## 本地开发

1. 克隆仓库：
```bash
git clone https://github.com/jianwang0212/yz-web.git
cd yz-web
```

2. 使用本地服务器打开：
```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx serve
```

3. 在浏览器中访问：http://localhost:8000

## 自定义

- 修改 `index.html` 更新内容
- 编辑 `styles.css` 调整样式
- 更新 `script.js` 添加交互功能
