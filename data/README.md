# Data Directory

This directory contains the website content data files.

## Files

- `content.json` - Original content data (92KB)
- `cleaned_content.json` - Cleaned and organized content data (207KB)

## Deployment

These data files should be deployed separately or loaded from an external source in production.

For local development, place the JSON files in this directory.

## Note

These files are excluded from git to keep the repository focused on core code. They should be:
1. Deployed separately to your hosting service
2. Or loaded from an external API/CDN
3. Or managed through a CMS
